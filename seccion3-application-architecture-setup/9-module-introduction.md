# 9. Module Introduction

En está segunda sección, vamos a estar construyendo el **splash wing** que vemos cuando iniciamos nuestra aplicación.

Cuando abrimos la verión completa de nuestra app de Flutter, podemos ver este **splash screen**.

La razón por la que la requerimos es poque nosotros tenemos ciertas cosas que nuestra aplicación necesita configurar antes que podamos
presentar la UI interactuando con la API.

Y la mejor forma es mostrar el **splash screen** en el background. Nosotros podemos hacer diferentes cosas como iniciar nuestros servicios,
quizás cargando todos los datos de nuestro `main.json`, en el cual tenemos información secreta haciéndola disponible a través de la aplicación.

Una vez iniciada, es tan simple como tener una pantalla cerca y navegar en la aplicación.
