# 4. Application Overview

Vamos a ver una introudcción rápida de la aplicación a desarrollar.

La apliación es **SPA** (*Single Page Application*), podremos:

* Buscar películas
* Tener categorías de películas, siendo las 2 siguientes:
  * *Popular*
  * *Upcoming*
* Tener más resultados conforme *scrolleamos* más
* Actuañizar la UI, haciendo clic en la iamgen de la película y actualizar el background de la aplicación con la misma

Para esto, usaremos la API de la web [TMDB](https://www.themoviedb.org/).
